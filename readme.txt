== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for 'Olena Blocks Set'
3. Activate the plugin from your Plugins page.

= From WordPress.org =

1. Download 'Olena Blocks Set'.
2. Upload the 'Olena Blocks Set' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
3. Activate 'Olena Blocks Set' from your Plugins page.
